## 核心页面
1. 登录和注册页 - 实现登录和注册功能
2. 首页 - 实现拍摄鸟类照片并识别功能
3. 详情页 - 鸟类信息查看功能
4. 社区页 - 实现分享和交流鸟类信息功能
5. 百科页 - 实现专业查询鸟类信息功能
6. 个人用户页 - 实现个人用户信息显示和编辑功能
## 交互说明
- 点击 注册按钮 跳转到 注册页面
- 点击 登录按钮 跳转到 首页
- 点击 社区按钮 跳转到 社区页
- 点击 百科按钮 跳转到 百科页
- 点击 个人用户按钮 跳转到 个人用户页面
- 点击 社区页中的鸟类图片 跳转到 鸟类分享详情页面
- 点击 百科页中的鸟类图片 跳转到 鸟类信息详情页面
- 点击 个人用户页面编辑按钮 跳转到 用户编辑界面
