AviAI鸟类智能观测微信小程序前端文档
1. 文档概述
本文档旨在为“AviAI鸟类智能观测”微信小程序的前端开发提供全面的技术指导与规范。该项目是一个基于人工智能的鸟类识别与观测工具，核心功能是利用手机摄像头实时检测并识别鸟类种类，并提供丰富的鸟类知识库供用户搜索和学习。

本项目采用 uni-app 框架进行开发，提高开发效率并降低维护成本。文档内容涵盖模块功能划分、技术选型理由、项目目录结构以及详细的运行方式，是开发团队进行编码、测试和上线的依据。

2. 模块功能
根据该项目需求，AviAI小程序前端主要划分为以下三个核心功能模块：

2.1 智能识别模块
这是应用的核心功能入口，旨在通过AI技术即时辨别鸟类。

摄像头调用：启动手机摄像头，提供实时预览画面。用户可对准目标鸟类进行拍摄。

图像上传与识别：支持从相册上传图片。将拍摄或选择的图片提交至后端AI服务进行分析。

结果展示：接收后端返回的识别结果，以卡片形式清晰展示鸟类的名称（包括中文名与学名）、科属分类、识别置信度以及特征描述。

识别记录：自动保存用户的每一次识别历史，方便用户回顾和管理自己的观测记录。

2.2 物种百科模块
该模块作为一个知识库，供用户浏览和学习鸟类知识。

分类浏览：支持按照鸟类科目（如雉科、鸭科、鹰科等）或地理分布对鸟类进行分类浏览。

物种详情页：点击具体鸟类后进入详情页，展示高清图鉴、形态特征、生活习性、叫声、保护等级等详细信息。

附近分布：基于用户地理位置，展示该鸟类在当地的分布情况或近期观测记录（此功能需后端支持）。

2.3 搜索与发现模块
帮助用户快速定位目标鸟类信息。

全局搜索：支持通过鸟类的中文名、英文名或别名进行模糊搜索，快速定位到目标物种。

筛选与排序：在搜索结果或百科列表中，支持按体型、颜色、栖息地等特征进行筛选，或按名称首字母排序。

热门推荐：在搜索页面展示当前热门或新近发现的鸟类，引导用户探索。

3. 技术选项
为了确保项目的开发效率、运行性能及可维护性，采用以下技术栈：

3.1 核心框架
uni-app 

选型理由：uni-app 基于 Vue.js 语法，学习成本低，开发效率高。其最大的优势在于“一套代码，多端发布”，不仅覆盖微信小程序，还能在未来需要时轻松编译为 H5 或 App，具有很强的扩展性。同时，uni-app 拥有丰富的插件市场和成熟的社区支持，能够快速解决开发中遇到的问题。

3.2 开发语言与样式
Vue 3 组合式 API ：采用 Vue 3 的组合式 API（Composition API）进行开发，相比传统的选项式 API，它能更好地组织逻辑代码，提高代码的复用性和可读性，特别是在处理复杂的识别业务逻辑时优势明显。

TypeScript ：引入 TypeScript 进行静态类型检查，可以显著降低运行时错误，提升大型项目的代码健壮性和可维护性。

CSS 预处理器：使用 SCSS 编写样式，利用其变量、混合等特性，配合 rpx 单位实现响应式布局，确保页面在不同屏幕尺寸的手机上都能完美适配 。

3.3 状态管理与网络请求
Pinia ：作为 Vue 官方推荐的状态管理库，Pinia 相比 Vuex 更加轻量、直观，且对 TypeScript 支持极佳，用于管理用户登录状态、识别记录等全局数据。

网络请求：封装 uni.request 方法，统一处理 API 接口调用、请求拦截、响应拦截以及错误处理。

3.4 UI 组件库
uView UI / uni-ui ：优先使用 uni-app 官方或成熟的第三方组件库（如 uView UI），避免重复造轮子，利用现成的导航栏、表单、卡片等组件快速搭建页面。

静态资源管理：全局图片、字体等放在 static 目录。分包特有的资源放在分包的 static 下，避免打包到主包中 。

5. 运行方式
以下是项目从拉取代码到在微信开发者工具中运行的全步骤。

5.1 环境准备
Node.js：安装 v16.15.0 或更高版本 。

包管理器：推荐使用 pnpm (v8.6.10+) 或 npm (v8+) 。

微信开发者工具：下载并安装最新版本的 微信开发者工具。

HBuilderX（可选）：虽然可以使用命令行开发，但下载 HBuilderX 可以更方便地使用 uni-app 的可视化运行菜单。

5.2 安装与运行
克隆项目

bash
git clone [https://github.com/hzy2005/AviAI]
cd frontend
安装依赖
建议使用国内镜像源加速。

bash
pnpm i --registry=https://registry.npmmirror.com
或

bash
npm install --registry=https://registry.npmmirror.com
启动开发服务
运行以下命令编译微信小程序版本，该命令会启动监听模式，代码变动时自动重新编译 。

bash
npm run dev:mp-weixin
或使用 HBuilderX：打开项目，点击菜单栏 运行 -> 运行到小程序模拟器 -> 微信开发者工具 。

导入开发者工具
启动微信开发者工具，点击导入项目。项目目录指向本项目下的 dist/dev/mp-weixin 文件夹。输入自己的小程序 AppID，即可预览和调试 。

5.3 调试与发布
调试：在微信开发者工具中，可以利用 Sources 面板（需根据 sourcemap 找到对应 webpack:// 下的源文件）打断点调试 JS，或使用 Console 查看日志 。对于 Vue 组件状态，可以使用 AppData 面板查看 。

发布：当开发完成需要打包上传时，执行以下命令进行生产环境编译：

bash
npm run build:mp-weixin
编译完成后，在微信开发者工具中打开 dist/build/mp-weixin 项目，点击 上传 按钮即可提交代码到微信公众平台审核 。
